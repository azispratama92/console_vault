# console_vault
Log catatan pembuatan Laptop sebagai console backup companion.

ConsoleVault\
├── Docs\
├── Tools\
├── Backups\

Saya berencana menggunakan 2 (dua) HDD @berkapasitas 1TB, 1 SSD untuk sistem operasi windows beserta tools, console mods. 

1 sisanya saya pergunakan untuk menyimpan backup game dari console-console yang saya miliki. 

Project ini masih possible dilakukan dengan ketersediaan, perangkat yang saya miliki.
